###Function Declaration VS Function Expression
- Function Declaration : 전체가 Hoisting 대상
- Function Expression : 변수만 Hoisting


###Function Scope & Execution Context
Scope
- (변수의)유효범위
- 함수가 정의될 때 결정

Execution Context
- 실행 컨텍스트
- 실행되는 코드 덩어리 (추상적 개념)
- 함수가 실행될 때 생성
